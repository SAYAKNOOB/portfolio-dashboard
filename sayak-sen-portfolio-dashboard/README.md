# Interactive Portfolio Dashboard

A professional, interactive, and responsive portfolio dashboard built with **React**, **Vite**, and **Tailwind CSS**. This application serves as a comprehensive digital resume designed specifically for an SDET (Software Development Engineer in Test), AI Automation Engineer, and Quality Analyst (QA). 

## 🌟 Overview

This portfolio goes beyond a traditional static resume by providing a dynamic, app-like experience. It features a sleek dual-theme (dark and light mode) interface that adapts to user preferences, paired with smooth animations powered by Framer Motion. 

Key highlights of the application include:
- **Interactive Terminal Console:** A developer-centric sandbox terminal that provides a unique and engaging way to explore the profile.
- **Categorized Sections:** Neatly organized tabs covering a detailed Skills Matrix, an Experience Timeline, Key Achievements, and Education & Certifications.
- **Functional Utilities:** Built-in tools such as a one-click PDF resume generator and a native browser print-ready layout, making it highly practical for recruiters and hiring managers.
- **Responsive Design:** Fully optimized for seamless viewing across all devices, from large desktop monitors to mobile screens, ensuring a consistent and polished experience.

Whether you are exploring the interactive tech stack, navigating through the experience timeline, or downloading the generated PDF, this dashboard provides a modern, fast, and highly visual representation of professional expertise.

## 🚀 Features

- **Dark & Light Mode:** Toggle seamlessly between themes with persistent user preferences.
- **Print & Download:** Generate a PDF version of the resume dynamically or trigger the native print dialog for a print-friendly view.
- **Animated Interactions:** Engaging micro-interactions and layout transitions.
- **Responsive Grid:** Complex layouts that adapt flawlessly to mobile, tablet, and desktop viewports.
- **Clipboard Integration:** Easy copying of contact details (email, phone) with visual feedback.

## 🛠️ Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Motion (Framer Motion)
- **PDF Generation:** jspdf

## 📦 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You will need **Node.js** (version 18 or higher is recommended) installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio-dashboard
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server with hot-module replacement (HMR), run:

```bash
npm run dev
```

The application will be available at `http://localhost:3000` (or another port if 3000 is occupied).

### Building for Production

To create an optimized production build, run:

```bash
npm run build
```
This will compile your TypeScript files and bundle your React application into the `dist` directory.

To preview the production build locally, run:

```bash
npm run start
# or npm run preview depending on your package.json scripts
```
